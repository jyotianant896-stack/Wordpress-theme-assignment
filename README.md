# Wordpress-theme-assignment
Custom WordPress theme with responsive UI including navbar, hero section, brands, and product listing.
Project Overview

This project is a custom WordPress theme developed as part of an assignment.
It focuses on creating a modern, responsive UI similar to an e-commerce website.

---

✨ Features

- Responsive Navbar with logo and icons
- Hero Section with heading, description, and "Shop Now" button
- Statistics Section (Brands, Products, Customers)
- Brand Logos Section
- New Arrivals (Product Listing)
- Clean and modern UI design

---

🛠️ Technologies Used

- HTML5
- CSS3
- PHP
- WordPress

---

🚀 How to Run

1. Install XAMPP / WAMP
2. Place the theme inside:
   "htdocs/wordpress/wp-content/themes/"
3. Activate the theme from WordPress dashboard
4. Open browser:
   "http://localhost/wordpress"
